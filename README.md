# 🌱 습도 데이터 이상 탐지 및 AI 학습 기반 분석 프로그램

## 📌 프로젝트 개요

스마트팜 환경에서 수집되는 습도 데이터를 기반으로  
AI 모델을 활용하여 이상 상태를 자동으로 탐지하는 프로그램입니다.

센서 데이터는 환경 제어 및 설비 상태 판단에 중요한 역할을 하며,  
이상 값이 발생할 경우 작물 생육 문제나 설비 이상으로 이어질 수 있습니다.

이 프로젝트에서는  
데이터 전처리 → AI 모델 학습 → 이상 탐지 → 결과 시각화  
과정을 하나의 흐름으로 구현하였습니다.

---

## 🧠 AI 학습 및 분석 방식

본 프로그램은 머신러닝 기반 이상 탐지 알고리즘인  
**Isolation Forest**를 사용하여 데이터를 학습합니다.

### ✔ 학습 흐름

1. CSV 형태의 습도 데이터 입력  
2. 결측치 제거 및 데이터 전처리  
3. Isolation Forest 모델 학습  
4. 데이터 분포를 기반으로 이상치 판단  
5. 정상 / 이상 데이터 분류  

👉 특징  
- 정상 데이터 패턴을 학습한 뒤  
- “다른 패턴”을 이상으로 판단하는 방식

---

## ⚙️ 전체 동작 구조

입력 데이터 → 전처리 → AI 학습 → 이상 판단 → 결과 출력

- 입력: 센서 기반 습도 데이터 (CSV)
- 처리: 데이터 정제 및 전처리
- 학습: Isolation Forest 모델 적용
- 판단: 이상치 여부 분류
- 출력: 결과 CSV + 시각화 그래프

---

## 📊 주요 기능

- 습도 데이터 전처리 (결측치 처리)
- AI 기반 이상치 탐지 (Isolation Forest)
- 데이터 분포 시각화 (산점도)
- 이상 여부 포함된 결과 파일 생성

---

## 🔗 자동화 / 설비 관점에서의 의미

센서 데이터를 기반으로 상태를 판단하고  
이상 여부를 자동으로 분류하는 구조를 구현하였습니다.

이는 실제 자동화 설비에서

- 센서 입력 → 상태 판단 → 제어 또는 경고

로 이어지는 구조와 유사하며,  
데이터 기반으로 설비 상태를 판단하는 흐름을 이해하는 데 목적이 있습니다.

이를 통해  

- 데이터 → 판단 → 결과 출력 구조 이해
