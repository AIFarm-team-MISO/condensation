# 스마트팜 습도 데이터 이상 탐지 및 AI 기반 분석 시스템

## 프로젝트 개요

스마트팜 환경에서 수집되는 습도 데이터를 기반으로  
이상 상태를 자동으로 탐지하는 머신러닝 기반 분석 프로그램입니다.

스마트팜에서는 온도, 습도, CO2 등의 센서 데이터가  
작물 생육과 설비 상태를 판단하는 핵심 요소로 활용됩니다.

특히 습도 이상은 결로 발생, 병해 증가, 설비 이상으로 이어질 수 있어  
빠른 감지와 대응이 중요합니다.

본 프로젝트에서는 이러한 문제를 해결하기 위해  
데이터 기반 이상 탐지 + 도메인 룰 기반 보정 구조를 적용하여  
실제 환경에 가까운 판단 흐름을 구현하였습니다.



## AI 학습 및 분석 방식

본 프로그램은 비지도 학습 기반 이상 탐지 알고리즘인  
Isolation Forest를 사용합니다.

### 학습 및 판단 흐름

1. CSV 형태의 센서 데이터 입력  
2. 결측치 및 이상값 처리 (데이터 전처리)  
3. Isolation Forest 모델 학습  
4. 데이터 분포 기반 이상치 탐지  
5. 도메인 조건을 반영한 최종 판단 보정  


### 핵심 특징

- 정상 데이터 패턴을 학습하여  
  → 패턴에서 벗어난 데이터를 이상치로 판단

- 별도의 정답(label) 없이도 이상 탐지 가능  
  → 실제 센서 데이터 환경에 적합


## 전체 동작 구조

- 입력: 습도 센서 데이터 (CSV)
- 처리: 결측치 및 이상값 정제
- 학습: Isolation Forest 적용
- 판단: 이상 여부 분류 + 룰 기반 보정
- 출력: 결과 CSV + 시각화 그래프

## 주요 기능

- 습도 데이터 전처리 (결측치 및 이상값 처리)
- 머신러닝 기반 이상치 탐지 (Isolation Forest)
- 도메인 조건 기반 판단 보정 (Rule-based filtering)
- 데이터 분포 시각화 (산점도)
- 이상 여부 포함된 결과 데이터 생성


## 도메인 룰 적용 (핵심 설계)

머신러닝 모델은 데이터 패턴 기반으로 이상을 판단하지만,  
실제 스마트팜 환경에서는 물리적인 조건이 함께 고려되어야 합니다.

본 프로젝트에서는 다음과 같은 조건을 추가하였습니다:
-> 습도 ≤ 60%는 이상치가 아닌 정상 상태로 처리

### 적용 이유

- 결로는 일반적으로 높은 습도에서 발생  
- 낮은 습도는 설비 이상이 아닌 정상 범위일 가능성이 높음  
- 모델의 과도한 이상 판단을 보정하기 위함  

이를 통해 AI 판단 + 환경 기반 조건을 결합한 현실적인 판단 구조를 구현하였습니다.


## 자동화 / 설비 관점에서의 의미

본 시스템은 다음과 같은 자동화 구조를 기반으로 합니다:


이는 실제 자동화 설비 및 PLC 시스템에서 사용되는 구조와 동일합니다.

- 센서 데이터 수집
- 조건 기반 상태 판단
- 이상 발생 시 알람 또는 제어

즉, 본 프로젝트는 데이터 기반 자동화 판단 구조를 이해하기 위한 기초 모델입니다.

## 기대 효과

- 센서 데이터 기반 이상 상태 조기 탐지
- 수작업 분석 대비 빠른 처리 가능
- 데이터 기반 의사결정 지원
- 반복 분석 작업 자동화

## 한계 및 개선 방향

### 현재 한계

- 단일 변수(습도) 기반 분석
- 데이터 간 상관관계 미반영 (온도, CO2 등)
- 실시간 처리 구조 미구현

### 향후 개선 방향

- 다변수 데이터 적용 (온도, CO2 등)
- 실시간 데이터 처리 시스템 구축
- 이상 탐지 결과 기반 자동 제어 연동
- PLC 및 설비 시스템과의 연결
